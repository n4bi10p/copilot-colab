# Demo Script & README Steps

## Demo Script
1. Start Supabase backend (`supabase start`).
2. Run VS Code extension in dev mode.
3. Sign in with Google/GitHub (Supabase Auth).
4. Create a new project and invite teammates.
5. Add tasks, update statuses, and chat in project room.
6. Show dashboard panel: task counts, recent messages, online users.
7. Demonstrate presence indicator and real-time sync.
8. Trigger AI feature (WBS generator or digest summary).
9. Show VibeLog panel for hackathon journey.

## README Steps
- Install dependencies: `npm install`
- Build extension: `npm run build`
- Start backend: `supabase start`
- Launch extension: VS Code Dev Mode
- Follow checklist for multi-user sync testing.
